# 🔍 Spam Detector

An SMS spam detection app built with Logistic Regression and a Streamlit chat interface.

## Features

- **Chat interface** — type any message and instantly get a spam/ham prediction with a confidence score
- **Word Cloud tab** — visualize the most frequent words in spam vs. ham messages from the training data
- **History tab** — track all analysed messages in the current session with a bar chart and log table
- **Sidebar** — live statistics (spam/ham counts, spam rate) and model info

## Model

| Property | Value |
|----------|-------|
| Algorithm | Logistic Regression |
| Features | TF-IDF (5000 tokens) |
| Accuracy | 96.77% |
| Dataset | [SMS Spam Collection](https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset) |

## Getting Started

**1. Install dependencies**
```bash
pip install pandas scikit-learn joblib streamlit wordcloud matplotlib
```

**2. Train the model**
```bash
python main.py
```

**3. Launch the app**
```bash
streamlit run app.py
```

Then open [http://localhost:8501](http://localhost:8501) in your browser.
